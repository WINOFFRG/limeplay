---
title: use-track-events
description: Utility hook for handling pointer events on slider tracks.
---

## Installation

```npm
npx shadcn add @limeplay/use-track-events
```

## Usage

Used internally by [`timeline-control`](/docs/components/timeline-control) and [`volume-control`](/docs/components/volume-control). Can be used to build custom sliders:

```tsx
import { useTrackEvents } from "@/hooks/limeplay/use-track-events"

function CustomSlider() {
  const { trackRef, thumbRef } = useTrackEvents({
    orientation: "horizontal",
    onPointerDown: (progress) => {
      console.log("Clicked at:", progress) // 0-1
    },
    onPointerMove: (progress, isDown) => {
      if (isDown) {
        console.log("Dragging at:", progress)
      }
    },
  })

  return (
    <div ref={trackRef} className="relative h-2 w-full bg-gray-200">
      <div ref={thumbRef} className="absolute h-4 w-4 rounded-full bg-blue-500" />
    </div>
  )
}
```

### Options

| Option | Type | Description |
|---|---|---|
| `orientation` | `"horizontal" \| "vertical"` | Slider direction |
| `onPointerDown` | `(progress, event) => void` | Click/touch start |
| `onPointerMove` | `(progress, isDown, event) => void` | Move/drag |
| `onPointerUp` | `(event) => void` | Release |
| `onPointerOut` | `(event) => void` | Pointer leaves |

`progress` is a normalized value (0–1) based on pointer position within the track element.
