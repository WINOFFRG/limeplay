---
title: use-seek
description: Utility hook for seeking forward or backward by a fixed offset.
---

## Installation

```npm
npx shadcn add @limeplay/use-seek
```

<Callout>
Requires [`timelineFeature`](/docs/hooks/use-timeline) registered in your [`createMediaKit`](/docs/components/media-provider).
</Callout>

## Usage

```tsx
import { useSeek } from "@/hooks/limeplay/use-seek"

function SeekButtons() {
  const { seek } = useSeek()

  return (
    <div>
      <button onClick={() => seek(-10)}>Back 10s</button>
      <button onClick={() => seek(-30)}>Back 30s</button>
      <button onClick={() => seek(10)}>Forward 10s</button>
      <button onClick={() => seek(30)}>Forward 30s</button>
    </div>
  )
}
```

`seek(offset)` adds the offset (in seconds) to the current time. Negative values seek backward. The value is clamped to `[0, duration]`.

This is an action-only hook — it doesn't track state. Timeline state is managed separately by [`timelineFeature`](/docs/hooks/use-timeline).
