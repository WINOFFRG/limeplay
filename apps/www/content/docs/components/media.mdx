title: Media Element
description: Native HTML media element (video or audio) connected to the store.
---

## Installation

```npm
npx shadcn add @limeplay/media
```

<Callout>
Requires [`mediaFeature`](/docs/concepts) registered in your [`createMediaKit`](/docs/components/media-provider).
</Callout>

## Usage

Renders a `<video>` or `<audio>` element and binds it to the media store automatically.

```tsx title="components/media-player.tsx"
import { Media } from "@/components/limeplay/media"

<Media as="video" className="size-full object-cover" />
```

Switch to audio:

```tsx
<Media as="audio" />
```

### Forwarding Refs

```tsx
const videoRef = React.useRef<HTMLVideoElement>(null)

<Media as="video" ref={videoRef} autoPlay muted />
```
